Navigate Windows with Your Keyboard

NavKey is a powerful keyboard navigation tool that helps you navigate your Windows computer entirely using the keyboard.

Get Started

About NavKey

NavKey empowers users to navigate and control their Windows computer using only the keyboard. Whether you're looking to increase productivity, reduce mouse usage, or improve accessibility, NavKey provides an intuitive keyboard-based interface for seamless computer navigation.

Designed with efficiency in mind, NavKey eliminates the need to switch between keyboard and mouse, keeping your hands on the keys for faster workflow management.

How to Use NavKey

1. Download & Run

Download the NavKey.exe file in the Release folder and run it on Windows 11.


If you are logged in as an administrator, you can run it normally. Otherwise, right-click the file and choose “Run as administrator.”


NavKey runs in the background and does not open a visible window.

2. Enter Command Mode

Press Ctrl to toggle between normal typing mode and Command Mode.

When Command Mode is active, the cursor will show a red indicator so you always know NavKey is listening.

3. Navigate Using Keys

NavKey works by entering a temporary Hint Mode, where the keyboard is used for navigation instead of the mouse.


  • S + D + F — Enter Hint Mode and show navigation hints
  • A–Z — Select a visible hint
  • Hold D + F — Scroll down
  • Hold J + K — Scroll up
  • G + H — Switch windows (Alt+Tab style)
  • Esc — Exit Hint Mode and hide all hints

Press Ctrl at any time to toggle between Insert Mode and Command Mode. When Command Mode is active, the cursor displays a red indicator.

Ctrl-based shortcuts always pass through, so standard shortcuts like copy, paste, and save continue to work normally.
